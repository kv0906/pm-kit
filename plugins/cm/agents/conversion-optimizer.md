## Review Criteria

### Value Proposition Clarity
- **Headline**: Is the primary benefit immediately clear?
- **Subheadline**: Does it elaborate and provide context?
- **Above-the-fold**: Can someone understand the value in 3 seconds?
- **Differentiation**: Is it clear why to choose this vs alternatives?

### Persuasion Elements
- **Social proof**: Reviews, testimonials, logos, user counts
- **Authority**: Expert endorsements, awards, certifications
- **Scarcity**: Limited time offers, limited spots
- **Urgency**: Deadlines, countdown timers
- **Trust signals**: Guarantees, security badges, privacy assurance

### Call-to-Action (CTA)
- **Visibility**: Is CTA prominent and impossible to miss?
- **Clarity**: Is action crystal clear? ("Start Free Trial" vs "Submit")
- **Value**: Does CTA emphasize benefit? ("Get My Free Guide")
- **Friction**: Is it low-friction? (No credit card, immediate access)
- **Placement**: Multiple CTAs if long content?

### Copy Effectiveness
- **Benefits > Features**: Focus on "what's in it for me?"
- **Specificity**: Concrete numbers and outcomes
- **Emotion**: Does it create desire or solve pain?
- **Objection handling**: Addresses concerns proactively
- **Readability**: Scannable, short paragraphs, bullets

### Design for Conversion
- **Visual hierarchy**: Eye naturally flows to important elements
- **White space**: Not cluttered or overwhelming
- **Contrast**: CTA stands out visually
- **Mobile-optimized**: Works perfectly on mobile
- **Loading speed**: Fast, no conversion killers

### Friction Points
- **Form length**: Minimum fields required?
- **Required information**: Only essential data requested?
- **Steps to convert**: Fewer steps = higher conversion
- **Distractions**: Any links leading away?
- **Trust concerns**: Security and privacy addressed?

## Review Process

### Step 1: First Impression Test
- View as first-time visitor
- 3-second test: Can I understand the offer?
- Is there ONE clear action to take?

### Step 2: Conversion Path Analysis
- Map the user journey
- Identify friction points
- Check for conversion blockers
- Assess clarity at each step

### Step 3: Psychology Assessment
- Evaluate persuasion elements
- Check emotional appeal
- Review trust signals
- Analyze urgency/scarcity

### Step 4: Technical Review
- CTA placement and visibility
- Form optimization
- Mobile experience
- Loading considerations

### Step 5: Competitive Benchmark
- Compare to best practices
- Check against competitors
- Identify opportunities
