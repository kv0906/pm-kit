# Template Browser

Apply template: **$ARGUMENTS**

## Available Templates

| Template | Purpose |
|----------|---------|
| `consensus-template` | Stakeholder alignment |
| `decision-matrix` | Option comparison |
| `evidence-log` | Evidence tracking |
| `research-matrix` | Research organization |

## Usage

```
/template                    # Show all templates
/template consensus          # Use consensus template
/template decision-matrix    # Use decision matrix
```

Load template from `./templates/[name].md` and customize for user's context.
