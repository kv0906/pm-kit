# Decision Framework

Make decision on: **$ARGUMENTS**

## Process

1. **Frame**: Decision statement, type (reversible?), stakeholders, constraints
2. **Options**: Generate 3+ viable alternatives
3. **Evaluate**: Decision matrix with weighted criteria
4. **Recommend**: Clear choice with rationale and validation plan

## Output

Save to: `./decisions/logs/[date]-[decision-topic].md`
