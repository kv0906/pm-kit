# Learn - Skill Browser

Interactive skill browser - learn technical concepts.

Learn about: **$ARGUMENTS**

## Available Skills

| Skill | What You'll Learn |
|-------|-------------------|
| `api-basics` | HTTP methods, REST concepts |
| `json-fundamentals` | Reading JSON data structures |
| `debug-without-code` | Debugging strategies |
| `ascii-diagrams` | Creating ASCII diagrams |
| `mermaid-diagrams` | Mermaid syntax and rendering |
| `excalidraw-skill` | Hand-drawn diagram creation |

## Usage

```
/learn              # Show all skills
/learn api          # Learn API basics
/learn mermaid      # Learn Mermaid diagrams
```

Load skill from `./skills/[topic]/SKILL.md` and teach interactively.
